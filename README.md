# TODO List App

This is a simple Todo List web application built using React.

## 🚀 Features
- Add new tasks
- Delete tasks
- Move tasks up and down

## 📁 Project Structure
- src/components → Contains UI components
- src/pages → Contains page-level components

## 🛠️ Technologies Used
- React
- JavaScript
- CSS

## ▶️ How to Run
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
