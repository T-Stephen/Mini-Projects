# Mini Projects Collection

This repository contains multiple beginner-friendly mini projects built using JavaScript and React.

## 🚀 Projects Included
- Calculator
- Clock
- Password Generator
- Todo List
- Temperature Converter
- Text Utils

## ✨ Features
- Simple UI
- Beginner friendly
- Easy to understand code

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- React

## ▶️ How to Run
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
